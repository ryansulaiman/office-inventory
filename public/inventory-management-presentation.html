Office Inventory
Management System

A real-time, role-based inventory system built to track, manage, and control all office devices and equipment — from request to return.

📱 Mobile Friendly ⚡ Real-Time 🔐 Role-Based Access ☁️ Cloud-Based
The Problem

What We Were
Dealing With Before

Managing office inventory manually was creating real operational issues.

📋
No tracking system
No clear record of who has which device or when it was assigned.
❓
Lost & unaccounted items
Devices go missing with no way to trace who last had them.
⏰
No return visibility
No way to know if borrowed items were returned or still out with staff.
🗂️
Manual paperwork
Requests done verbally or on paper — easy to lose and hard to track.
🔄
No transfer process
Staff passing devices to each other with no record or approval process.
📊
No stock overview
Management had no real-time visibility into what's available or in use.
The Solution

A System Built for
How We Actually Work

A purpose-built inventory app that gives everyone exactly what they need — nothing more, nothing less.

📱
Unit-level tracking
Every phone, laptop, and printer has its own unique code (e.g. PHN-001) tracked to the person holding it.
✅
Request & approval flow
Staff submit requests. Assistants assign exact units. Full paper trail every time.
📅
Return date & overdue alerts
Every assignment has a due date. Overdue items are flagged automatically in red.
🔄
Peer transfers
Staff can transfer devices to workmates with a formal accept/decline process.
⚠️
Incident reporting
Damaged or lost devices are reported and tracked through to resolution.
📊
Live dashboard
Real-time overview of all stock levels, overdue items, and open incidents.
Access Control

3 Roles, Clear Responsibilities

Each person sees only what they need. No confusion, no unauthorized access.

Admin

Full Control

  • Full access to everything
  • Manage users & roles
  • View audit log
  • Edit & delete inventory
  • Resolve incidents
  • Return all items for any staff
Inventory Assistant

Day-to-Day Ops

  • Approve & assign requests
  • Assign specific unit codes
  • Process returns
  • Approve extensions
  • Report & resolve incidents
  • Add inventory & units
Staff

Self-Service

  • Request devices & items
  • View own assigned items
  • Transfer to workmates
  • Request return extensions
  • Report damaged/lost devices
  • Accept or decline transfers
Workflow

How a Request Works

From request to return — every step is tracked and logged automatically.

👤
Staff Requests
Picks item, quantity & expected return date
→
🔔
Assistant Reviews
Sees pending request in dashboard
→
📱
Unit Assigned
Picks exact unit e.g. PHN-001 and approves
→
📅
Due Date Set
Return date tracked, overdue alert if missed
→
↩️
Returned
Assistant processes return, stock restored

Every action — requests, approvals, returns, incidents, transfers — is recorded in a full Audit Log with the name, date, and details. Nothing is ever lost.
